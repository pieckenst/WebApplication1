Privacy Policy

Learn more about how data is collected and used by Windows Update Restored.

Last Updated: 04/17/2024

 Collected data
General Data Collection Policy

We do not collect any data. The one used to check for updates is not accessible by Windows Update Restored and only stays on your local machine. Collecting personal data is something that no one on the Windows Update Restored team likes and wants, so we do not collect anything. You won't have the feeling of being spied on when using Windows Update Restored. As we're hosting the websites on our own, no data or requests go to Microsoft Corporation.
 Data used by Windows Update when checking for updates
Data Evaluated Locally During Update Audits

This is what Windows Update uses when checking for updates:

  • The current Microsoft software installed on your computer (only Windows products, like Windows Media Player, etc.);
  • The name and version number of the Windows operating system you're checking for updates on;
  • The version numbers of the Windows software installed on your computer.

That's all. Windows Update does not use anything else to provide you the latest updates for your computer. This is available for every version of Windows Update & Microsoft Windows. Please also note that all this data is only used locally, and nothing is being sent to Windows Update Restored. All the files downloaded from Windows Update Restored to check for updates are downloaded in C:\Program Files\WindowsUpdate and C:\Program Files\WindowsUpdate\V4.
 Advertising and cookies
Advertising and Monetization Disclaimers

Nothing is collected at all. We are completely against advertisements or anything related to selling personal data to third-party companies of any kind. We do not make any profits in general, and the server costs are generously paid by millennium_tech, which is the server owner.
 Website logs
Standard Web Server Logging

They are kept for 14 days. The Windows Update Restored websites use industry-standard web server logging. They are deleted from the server after 14 days and contain your user agent, as well as your IP address. They also contain the path on the website you were accessing and might contain the URL as well (but not all the time). Windows Update Restored does not use any of the logged data in any kind, except for assistance purposes, which are requested by the user on either Discord, MSFN, or email.
 Legacy Update
Legacy Update Proxy Connections

Visit the Legacy Update website. Windows Update Restored v5 uses the Legacy Update proxy server to check for and retrieve available updates. Please visit Legacy Update's Privacy Policy at http://legacyupdate.net/privacypolicy to learn more about what data is collected and for which purposes.
 Internet Explorer installers
Internet Explorer Installation Files

They only download original Microsoft files. All they do is download the original Microsoft signed files from our servers to keep them safe. They are not modified by anyone.
