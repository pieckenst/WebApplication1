Instructions

  1. Once the file has been downloaded and executed, it will start extracting the necessary DLLs and install the Visual Basic 5.0 runtime.
  2. Assuming everything was done successfully, you will get a window with various options. Pick the one corresponding to your needs. This software allows you to both check for inventories on the computer you're running it on, and on external hard drive(s).
  3. If you have selected your own machine, click the "Scan my computer" button. The software will then scan your machine for inventories, and give you a series of prompts and information if it finds anything.
    If you chose to scan an external hard drive, you will have to specify where to save the found files.
  4. Once the files have been saved, you can compress the folder (ZIP, RAR, 7Z) and send it to us through File Upload, Discord or email!

Known Issues

One issue has been found with this download.

File "VB5FR.DLL" is missing (VERSION 1.0.3 AND OLDER ONLY).
This happens because the Inventory Checker was compiled using a French version of the Microsoft Visual Basic 5.0 IDE. To fix this, extract VB5FR.DLL from the Inventory Checker SFX archive, copy it and paste it in %systemroot%\System32 (Windows 2000, XP, Server 2003) and %driveletter%\Windows\System (Windows 95, 98, 98SE, Me).

Version 1.0.4 now relies on Visual Basic 6. Check your version of the Inventory Checker using the "?" button in the application.
