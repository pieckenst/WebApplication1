 History of Version 4
Released alongside Windows XP in 2001, Windows Update V4 adopted the famous "Luna" theme and expanded its support to Windows 98, Me, and 2000 by 2002. This version replaced proprietary formats with an XML and ASP-based inventory system, though the server-side nature of ASP made the site significantly harder to restore. Notably, V4 was the first to drop support for Windows 95 and NT 4.0 while introducing the Windows Update Catalog, the direct predecessor to today's Microsoft Update Catalog.
 Supported Systems & Requirements
Compatible Operating Systems: Windows 98 / 98 SE, Windows Me, Windows 2000, Windows XP (to SP2), and Windows Server 2003.

Minimum Browser: Internet Explorer 5.0 (IE 5.5 or 6.0 Recommended).

Localization: Currently supporting 25+ languages (See the full list on Windows Update V4).
 Known Issues
Before using Windows Update Restored V4, here's what you need to know:
  • On x64 systems (AMD64): Windows Update will not install the ActiveX control. To fix this, you need to install the Unicode certificate first before using the site. Once done, Windows Update will work. On x64 systems, the V4 site won't find any updates, but it can be useful when you need to use the Windows Update Catalog.
  • On Windows XP Service Pack 3: The website does not show updates. Windows Update V4 only contains updates up to Windows XP Service Pack 2. Upgrade to Windows Update Restored V5 or V6 to continue receiving updates.
  • On Windows 98/98SE/Me: I get a few to no updates. We're lacking inventories for multilanguage support. Please check C:\Program Files\WindowsUpdate on Windows 9x/Me installations from pre-2011 to see if they may contain files we need. If that is the case, run the Inventory Checker first BEFORE using Windows Update Restored!
