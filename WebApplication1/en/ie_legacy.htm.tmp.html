Internet Explorer Downloads

This page lists downloads of Microsoft Internet Explorer that you can use with Windows Update.

Only Web Installers (except for 3.0x) are listed here, which download the required files from the Windows Update Restored servers.

Internet Explorer 4.0x and higher: If you wish to make offline installers for personal use, use the "Download only" mode.
